
# üçï Pizza Sales SQL Analysis
SQL-based data analysis project using pizza sales dataset


## Overview
This is my first data analytics project where I analyzed pizza sales data using SQL.
The project focuses on extracting business insights such as revenue trends,
popular pizza types, and customer ordering behavior.

## Tools Used
- MySQL
- MySQL Workbench
- GitHub
- PowerPoint

## Dataset
The dataset consists of four tables:
- orders
- order_details
- pizzas
- pizza_types

## Analysis Performed
### Basic
- Total orders
- Total revenue
- Highest priced pizza
- Most common pizza size
- Top 5 pizzas by quantity

### Intermediate
- Category-wise sales
- Orders by hour
- Average pizzas sold per day

### Advanced
- Revenue contribution percentage
- Cumulative revenue
- Top pizzas by category using window functions
